**Overview**
This project analyzes **Job Cloud advertising data (January 2019)** using **Power BI** to help **Metro Staffing** 
optimize its monthly job advertising budget. The analysis focuses on vendor efficiency, spend performance over time, 
and job category effectiveness.

## Objective
* Identify **high-performing advertising vendors**
* Evaluate **cost efficiency using CPA (Cost Per Application)**
* Analyze **advertising spend vs job applications over time**
* Recommend **data-driven budget optimization strategies**

## Dataset
* Source: Job Cloud Output ‚Äì January 2019
* Size: ~34,000 records
* Key fields: Date, Vendors, Category, Spend, Applies, CPA, CPC
